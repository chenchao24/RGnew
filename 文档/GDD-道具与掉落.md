## 2. 掉落规则

### 2.1 掉落判定流程

```
怪物死亡:
  1. 必掉: 经验宝石 (100%)
     - 经验值 = 怪物基础经验 × 波次经验倍率
  2. 概率掉落: 特殊道具（仅特殊怪有概率）
     - roll = random(0, 1)
     - 按掉落表判定
  3. 生成位置: 怪物死亡位置

特殊怪掉落表（多道具可同时掉落）:
  ├── 生命恢复 30%
  ├── 攻击力UP 15%
  ├── 移速UP 15%
  ├── 剑数+1  10%（仅坦克怪）
  └── 护盾    20%（仅自爆怪）
```

### 2.2 自动拾取

```
每帧检测:
  for each 地面道具:
    if distance(player, item) ≤ 拾取范围(30px):
      拾取道具 → 立即生效

注意：
  - 自动拾取无需按键
  - 拾取范围可通过 Buff 扩展（后期扩展）
  - 道具在地面最多存活 30 秒，超时消失
```

## 3. 道具效果详解

### 3.1 经验宝石

```
效果: 获得经验值
经验值 = monster.baseExp × wave.expMultiplier

例: 第4波击杀坦克怪
  = 12 × 1.3 = 15.6 → 16 经验（向上取整）
```

### 3.2 生命恢复

```
效果: 立即恢复 15 HP
规则:
  - 不超过 HP 上限
  - HP 满时拾取不浪费（但道具仍消耗）
  - 不受任何加成影响
```

### 3.3 攻击力UP

```
效果: 伤害 × 1.5
持续时间: 8 秒
规则:
  - 重新拾取时刷新持续时间（不会叠加倍数）
  - 乘法计算：最终伤害 = 基础伤害 × 1.5 × 其他加成
  - 有视觉提示：角色红色光晕
```

### 3.4 移速UP

```
效果: 移速 × 1.3
持续时间: 8 秒
规则:
  - 重新拾取时刷新持续时间
  - 乘法计算：最终移速 = 基础移速 × 1.3 × 其他加成
  - 有视觉提示：角色黄色尾迹变长
  - 不超过移速硬上限(500px/s)
```

### 3.5 剑数临时+1

```
效果: 旋转剑数量 +1（最多叠加3把临时剑）
持续时间: 10 秒
规则:
  - 每把临时剑独立计时（不是刷新）
  - 同时最多 3 把临时剑生效
  - 可与永久剑数量叠加
  - 临时剑到期后消失，保留最早获得的
  - 视觉: 临时剑为半透明
```

### 3.6 护盾

```
效果: 抵消下一次受到的完整伤害
持续时间: 15 秒
规则:
  - 重新拾取时刷新持续时间（不叠加层数，保持1层）
  - 抵消伤害后护盾消失
  - 视觉: 角色外围青色光环
  - 护盾期间受击 → 护盾破碎动画 + 0 HP损失
```

## 4. 道具生命周期

### 4.1 状态机

```
[ 生成 ] → [ 地面闲置 ] → [ 被拾取 ] → [ 销毁 ]
                │
                ├── 30秒超时 → [ 销毁 ]
                │
                └── 每帧闪烁动画
                    (剩余时间 < 5s 时加速闪烁)
```

### 4.2 地面存在时间

```
道具类型     存在时间   过期前闪烁
─────────────────────────────────
经验宝石     30s        5s
生命恢复     30s        5s
攻击力UP     20s        5s
移速UP       20s        5s
剑数+1       20s        5s
护盾         25s        5s
```

## 5. 道具经济平衡

### 5.1 每波预期掉落

```
第1波(15只普通小怪):
  经验: 15 × 5 = 75 经验 ≈ Lv3-Lv4
  道具: 0（无特殊怪）

第3波(25只, 30%特殊怪 ≈ 8只特殊怪):
  经验: 17×5 + 8×特殊怪经验 ≈ 85+ 约 150 经验 ≈ Lv6
  道具: 约 2-3 个生命恢复, 1-2 个攻击力UP, 1-2 个移速UP

第6波(45只, 60%特殊怪 ≈ 27只特殊怪):
  经验: 约 300+ 经验 ≈ Lv12-Lv14
  道具: 约 8 个生命恢复, 4 个攻击力UP, 4 个移速UP, 2 个剑数+1, 2 个护盾
```

### 5.2 设计原则

- **经验充足**：确保玩家在第一关能到达 Lv12-Lv16
- **恢复有限**：生命恢复不会让玩家无敌，需要配合走位
- **临时Buff有价值**：攻击力UP和移速UP在高压波次(5-6波)是救命稻草
- **特殊道具稀有**：剑数+1和护盾让击杀坦克/自爆有独特价值
