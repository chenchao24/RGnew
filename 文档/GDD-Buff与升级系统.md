## 1. 经验与升级曲线

### 1.1 升级公式

```
ExpToLevel(N) = Σ[k=2..N] (5k + 10)   (N ≥ 2)
ExpToLevel(1) = 0

即: 第N级所需累计经验 = 从第2级到第N级每级需求之和
```

```

───────────────────────────────────────────
```

## 3. 三选一算法

### 3.1 候选生成流程

```
每次升级，生成3个Buff候选：

Step 1: 关联槽（1个）
  从"玩家已持有Buff的关联系"中随机选1个Buff
  关联系定义：
    武器系 ↔ 元素系（输出联动）
    生存系 ↔ 控制系（防御联动）
    武器系 ↔ 生存系（通用联动）
  该Buff出现权重 +30%

Step 2: 随机槽（2个）
  从全部Buff池中完全随机选2个

Step 3: 汇总去重
  3个候选汇总，去重后若不足3个则补充随机Buff

Step 4: 品质分配
  按当前等级的品质概率分配每个候选的品质

Step 5: 排除已满
  移除已达到上限或已持有的唯一类Buff
  若移除后不足3个，重新生成
```

### 3.2 排除规则

- **上限排除**：已达到可叠加上限的 Buff 不出现在选项中
- **唯一排除**：锋利剑刃、钢铁意志、复活、斩杀 — 持有后不再出现
- **系别保障**：若某系所有 Buff 都满了，关联槽换系

## 5. 流派设计

### 5.1 三大流派

```
纯输出流（武器系为主）
  路线：剑刃数量 → 伤害强化 → 暴击率 → 旋转加速 → 锋利剑刃
  优势：清怪效率极高，Boss 战时间短
  劣势：脆皮，第5-6波容易被远程+快速怪秒杀
  通关关键：保持移动，用伤害碾压，避免被包围

元素控制流（元素系 + 控制系）
  路线：冰霜附魔 → 击退 → 雷电附魔 → 击退强化 → 元素精通
  优势：怪物难以近身，安全稳定
  劣势：清怪速度中等，Boss 战需要耐心
  通关关键：利用减速/击退创造空间，元素伤害稳定输出

吸血坦克流（生存系 + 武器系）
  路线：生命恢复 → 吸血 → 护盾生成 → 伤害强化 → 钢铁意志
  优势：容错率极高，可以适度站桩
  劣势：伤害中等，第6波可能清不完
  通关关键：平衡输出和生存，确保能处理第6波大量怪物
```

### 5.2 流派平衡原则

> 三种流派都应该能通关第一关。没有"永远正确"的选择——玩家的操作风格（好走位 / 偏站桩 / 喜欢控制）决定最适合的流派。

## 6. 升级UI规范

### 6.1 三选一界面

```
升级暂停游戏 → 屏幕中央出现3张Buff卡片

卡片布局：
┌──────────────────────────────────────┐
│                                      │
│     ┌──────┐  ┌──────┐  ┌──────┐   │
│     │ 图标  │  │ 图标  │  │ 图标  │   │
│     │Buff名 │  │Buff名 │  │Buff名 │   │
│     │ 品质  │  │ 品质  │  │ 品质  │   │
│     │ 描述  │  │ 描述  │  │ 描述  │   │
│     │ [选择]│  │ [选择]│  │ [选择]│   │
│     └──────┘  └──────┘  └──────┘   │
│                                      │
│     选择后游戏继续 (或按数字键1/2/3)  │
└──────────────────────────────────────┘

键盘快捷键: 1=左卡, 2=中卡, 3=右卡
```

### 6.2 Buff持有显示

- 屏幕底部或侧边显示当前持有的 Buff 图标列表
- 悬停显示 Buff 名称和效果
- 按 Tab 查看详细 Buff 面板
