# Buff 图标

每个 Buff 一个图标文件（建议尺寸：24x24，透明背景）

命名规范：使用 buff id 命名

- `sword_count.png` - 剑刃数量+1
- `rotation_speed.png` - 旋转加速
- `attack_range.png` - 攻击范围UP
- `damage_up.png` - 伤害强化
- `crit_chance.png` - 暴击率UP
- `multi_target.png` - 多重打击
- `sharp_blade.png` - 锋利剑刃
