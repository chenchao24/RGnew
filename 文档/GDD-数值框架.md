## 2. 闪避机制

### 2.1 判定流程

```
怪物攻击判定命中 → 随机数 roll [1, 100]
  ├── roll ≤ 闪避率 → 闪避成功
  │   ├── 播放角色半透明闪烁动画
  │   ├── 显示 "MISS" 浮动文字
  │   └── 获得 0.3s 无敌帧（防止多怪同时攻击秒杀）
  │
  └── roll > 闪避率 → 受到完整伤害
      ├── 显示红色伤害数字
      └── HP -= 怪物伤害
```

### 2.2 设计规则

- **独立事件**：每次受击独立判定，不使用伪随机分布（保持刺激感）
- **不累积保底**：不会因为连续未闪避而强行触发
- **无敌帧唯一**：0.3s 内不会受到任何伤害（同一帧多个怪物攻击只吃第一下）
- **硬上限**：闪避率无论如何叠加不超过 75%

```
─────────────────────────────────────────────────────────────────────

```

### 额外行为

| 怪物类型 | 特殊行为 |
|---------|---------|
| 小怪(普通) | 直线追踪，无特殊行为 |
| 小怪(快速) | 直线追踪，移速快但 HP 低 |
| 小怪(坦克) | 追踪 + 每 5s 向玩家冲刺一次(350px/s, 持续0.5s) |
| 自爆怪 | 追踪玩家，距离 ≤ 30px 时开始 3s 自爆倒计时。自爆造成 60px 范围 AOE 伤害。可以被击杀阻止自爆。 |
| 远程怪 | 距离玩家 ≤ 120px 时停步射击(弹道 300px/s)，距离 > 120px 时追踪接近。射击间隔 1.5s。弹道可被闪避。 |

### 怪物 AI 每帧逻辑

```
每帧(60fps, deltaTime ≈ 0.016s):
  1. 计算到玩家位置的单位方向向量
  2. position += direction × speed × waveSpeedMultiplier × deltaTime
  3. 检测与每把旋转剑的碰撞 → 触发伤害判定
  4. 检测与玩家距离：
     ├── 距离 ≤ attackRange → 进入攻击状态
     │   ├── windupTimer -= deltaTime (前摇0.3s)
     │   ├── 前摇结束 → 判定命中 → 闪避roll → 扣血
     │   └── cooldownTimer = 0.5s (后摇)
     └── 距离 > attackRange → 继续追踪
```

## 4. 波次强度递进

```
波次    怪物总数   特殊怪比例   HP倍率   伤害倍率   速度倍率   经验倍率
────────────────────────────────────────────────────────────────
第1波   15        0%          1.0x     1.0x       1.0x       1.0x
第2波   20        15%         1.2x     1.1x       1.05x      1.1x
第3波   25        30%         1.5x     1.2x       1.1x       1.2x
第4波   30        40%         1.8x     1.35x      1.15x      1.3x
第5波   38        50%         2.2x     1.5x       1.2x       1.4x
第6波   45        60%         2.8x     1.7x       1.25x      1.5x
```

> 特殊怪 = 快速怪 + 坦克怪 + 自爆怪 + 远程怪，按比例从普通小怪中替换。

### 波次刷新逻辑

```
第N波刷新时:
  1. totalCount = 波次怪物总数
  2. specialCount = floor(totalCount × 特殊怪比例)
  3. normalCount = totalCount - specialCount
  4. 在画面边缘随机位置生成(距玩家 > 200px 的安全生成距离)
  5. 特殊怪类型均匀分配: fast/tank/bomber/ranged 各约 specialCount/4
```

## 5. 平衡性校验

### 5.1 第1波 — 应偏简单

```
场景：1把剑，伤害10，转速180°/s，半径80px
怪物：15只小怪，HP=20

剑线速度 ≈ 2π×80 × (180/360) = 251px/s 弧长覆盖
每只怪需2次命中 → 理论清完15只需 8-12秒纯输出
加走位 → 实际约 25-35秒

结论: 剩余5-15秒拾取 → 偏简单 ✓ (建立信心)
```

### 5.2 第6波 — 应有压力

```
场景：假设 Lv12, 3-4把剑, 伤害~20, 转速~240°/s
怪物：45只, HP倍率2.8x, 伤害倍率1.7x

小怪HP = 20×2.8 = 56 → 需3次命中
坦克HP = 50×2.8 = 140 → 需7次命中
特殊怪 = 27只（快速怪速度187px/s 很快近身）

结论: 必须配合控制Buff(击退/减速)走位 → 站桩必死 ✓
```

### 5.3 Boss 战

```
Boss HP = 500, 伤害 = 20
假设 Lv15, 4-5把剑, 伤害~30, 有控制Buff

理论输出时间: 500÷(30×4) ≈ 4秒纯理论
实际因走位+Boss技能干扰 → 约 40-80秒激战

阶段3狂暴召唤小怪 → 强制玩家处理add → 需要兼顾走位和清小怪
```

## 6. 伤害计算公式

### 6.1 基础伤害

```
实际伤害 = 基础伤害 × (1 + 伤害加成%)
          × (暴击时 × 2.0)
          × (元素附魔额外计算，不参与暴击乘算)

伤害加成来源：伤害强化Buff(+5每层)、锋利剑刃(+10)
暴击判定：独立于普通伤害，先判定暴击再计算伤害
```

### 6.2 元素伤害

```
火焰: 命中后3秒内每秒3点 → 总9点额外伤害（不叠加，刷新时间）
冰霜: 减速30%持续2秒（不叠加，刷新时间）
雷电: 命中后弹跳3目标，50%伤害 → 理论最大溅射75%额外
毒液: 命中后6秒内每秒2点，可叠5层 → 最大60点额外伤害

元素伤害 = 独立于物理伤害，不受暴击影响
元素精通: 所有元素持续时间 × (1 + 50% × 层数)
```

### 6.3 闪避与减伤

```
最终受到伤害 = 怪物原始伤害
              × (1 - 减伤%)
              （钢铁意志: base 20%, HP<30%时 40%）

闪避判定在减伤之前，先判定闪避成功则不受任何伤害。
```
