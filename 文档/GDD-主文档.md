## 1. 游戏概述

**类型**: Survivors-like / 网页割草生存

**一句话描述**: 鼠标控制角色走位，自动旋转的剑群是唯一攻击手段。击杀怪物 → 拾取道具 → 升级选 Buff → 构筑流派 → 撑过 6 波怪物 + 1 个 Boss 即通关。

**技术栈**: HTML5 Canvas / PixiJS，纯网页运行，无后端依赖。

**目标体验**:
- 割草的爽快感（剑碰到怪物 = 即时伤害反馈）
- 构筑的深度（每次三选一 Buff，没有永远正确的选择）
- 压力的节奏（每 40 秒一波，紧张与喘息交替）

## 2. 设计支柱

> 所有设计决策都用这 4 条支柱检验。违反其中任何一条的设计都需要重新审视。

| # | 支柱 | 含义 | 检验标准 |
|---|------|------|----------|
| **P1** | **割草爽感** | 旋转剑群碰到怪物应有"切黄油"的即时反馈 | 中期一波怪清完，玩家嘴角会上扬 |
| **P2** | **选择有意义** | 每次升级选 Buff 时，没有"永远正确"的选项 | 不同流派（攻速/元素/吸血）都能通关 |
| **P3** | **压力曲线清晰** | 每 40 秒一波的节奏让玩家始终有"刚好撑住"的紧张感 | 不以"数值碾压"为唯一解法 |
| **P4** | **操作简单，深度在构建** | 鼠标移动是唯一操作，但 Buff 组合决定成败 | 第一次玩和第十次玩的决策完全不同 |

## 3. 核心玩法循环

### 3.1 三层循环

```
【秒级循环 0-5s】
  鼠标移动角色 → 旋转剑自动攻击 → 怪物受击
  (伤害数字/击退/元素特效) → 掉落道具 → 自动拾取

【波次循环 40s】
  第N波刷新 → 怪物聚拢追踪 → 角色走位+击杀
  → 掉落道具 → 波次清完 / 残存处理

【成长循环 持续全程】
  拾取经验宝石 → 累积经验 → 升级
  → 三选一 Buff → 构筑流派 → 应对更强怪物 → 击杀Boss
```

### 3.2 玩家核心决策

> "我在画面上的**位置**，决定了旋转剑的**打击面覆盖**，决定了哪些怪物先被清掉。
> 我选的每一个 Buff，都在改变'走位'和'站桩'的平衡点。"

## 4. 操作与输入

| 操作 | 输入 | 说明 |
|------|------|------|
| 角色移动 | 鼠标移动 | 角色始终跟随鼠标位置平滑移动 |
| 暂停 | Esc / P 键 | 暂停时可查看当前 Buff 列表 |
| 升级选择 | 鼠标点击 | 三选一 Buff 卡片点击确认 |

**无需键盘控制移动**——鼠标位置即角色目标位置。

## 5. 画面与视觉

### 5.1 画布规格

| 属性 | 值 |
|------|-----|
| 画布尺寸 | 1200 × 800 px |
| 帧率目标 | 60 FPS |
| 背景 | 暗色网格地面 + 微妙的环境光 |

### 5.2 视觉元素

| 元素 | 视觉表现 | 说明 |
|------|---------|------|
| 角色 | 发光圆点(半径12px) + 尾迹 | 直观体现移动方向 |
| 旋转剑 | 围绕角色的短线/刀刃 | 半径=攻击距离，可看到剑的轨迹弧 |
| 怪物 | 不同颜色的几何形 | 小怪=小圆, 坦克=大方块, 快速怪=三角, 远程=菱形 |
| 道具 | 发光的宝石图标 | 经验=蓝色, HP=绿色, 攻速=红色, 移速=黄色 |
| 伤害数字 | 浮动白色小字 | 暴击=金黄色加大, 元素=对应颜色 |
| 闪避 | "MISS"灰色文字 + 角色半透明闪烁 | 0.3s 无敌帧视觉反馈 |
| HP 条 | 屏幕顶部居中的红色血条 | 随血量变化有动画过渡 |
| 波次提示 | 屏幕中央大字 + 倒计时 | 每波开始/结束时显示 |

## 6. 游戏流程

### 6.1 完整流程

```
开始界面（标题 + "点击开始"）
  │
  ▼
第一关：6波普通怪物 + Boss
  │
  ├─ 第1波 (0s-40s)    → 玩家发育期
  ├─ 第2波 (40s-80s)   → 构建起步
  ├─ 第3波 (80s-120s)  → 压力出现
  ├─ 第4波 (120s-160s) → 流派成型
  ├─ 第5波 (160s-200s) → 高压走位
  ├─ 第6波 (200s-240s) → 巅峰压力
  └─ Boss战 (240s起)   → 终极考验
  │
  ├─ 胜利 → 结算界面（击杀数/时间/等级/评价）
  │
  └─ 失败(HP=0) → 结算界面 → 重新开始
```

### 6.2 波次间过渡

- 当前波次怪物全部清完 → 立即进入下一波倒计时
- 若在 40s 倒计时结束前清完 → 剩余时间用于拾取道具 + 走位准备
- 40s 倒计时到 → 无条件刷出下一波（未清完的怪物继续存在）

### 6.3 通关条件

- 击杀第 6 波所有怪物 + 击败 Boss = **通关**
- 通关后显示结算画面

### 6.4 失败条件

- 角色 HP ≤ 0 = **游戏失败**
- 显示结算画面（本次获得的最高等级/击杀数/存活时间）

## 7. 后续扩展路线

### v1.1 — 短期

- 2 个新角色（不同初始属性：高移速低 HP / 高伤害低速）
- 关卡选择界面
- 本地击杀统计排行榜

### v1.2 — 中期

- 局外永久升级系统（金币购买初始属性加成）
- 3 个新 Buff 系：光系 / 暗系 / 召唤系
- 无尽模式

### v2.0 — 长期

- 双人合作模式
- 自定义关卡编辑器
- 赛季挑战（随机 Ban 某些 Buff，强制新流派）
